## Installation

1. Clone the repo

   ```sh
   git clone https://github.com/ntegrals/cryptoreach
   ```

2. Head over to [the underlying SaaS template](https://github.com/vercel/nextjs-subscription-payments) to set up Supabase and Stripe.


3. Install the dependencies
   ```sh
   npm install
   ```
4. Run the app
   ```
   npm run dev
   ```
5. Deploy to vercel
